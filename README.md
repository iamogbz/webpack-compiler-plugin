Easily listen to webpack compiler hooks and execute commands on events

## Shell Commands

```sh
npm run test      # run jest tests
```

```sh
npm run commit    # run commitizen
```

```sh
npm run build     # webpack build
```

### Typescript

```sh
make typescript
```
